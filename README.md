<h2>📌 Overview</h2>

<p>
This project performs Exploratory Data Analysis (EDA) on an employee salaries dataset using Python.  
The goal is to extract insights about salaries, job titles, overtime pay, and employee compensation trends.
</p>

<hr>

<h2>🛠️ Tools Used</h2>

<ul>
  <li>🐍 Python</li>
  <li>🐼 Pandas</li>
</ul>

<hr>

<h2>📂 Dataset</h2>

<p>The dataset contains employee salary information including:</p>

<ul>
  <li>👤 Employee Name</li>
  <li>💼 Job Title</li>
  <li>💰 Base Pay</li>
  <li>⏱️ Overtime Pay</li>
  <li>🏥 Benefits</li>
  <li>📊 Total Compensation</li>
  <li>📅 Year</li>
</ul>

<ul>
